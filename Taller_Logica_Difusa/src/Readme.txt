Las dos variables de entrada son presupuesto y calidad:
En el programa se maneja el valor de presupuesto de 10 a 400, representando un presupueso de 10,000 a 400,000 como ejemplo de presupuesto para la compra de un pantalon.
La representación de calidad se da como un valor entre 0 y 5, representando la calificación usualmente usada en sitios de compra en línea con un sistema de "estrellas".

La variable de salida es la probabilidad de compra: un valor entre 0 y 100
